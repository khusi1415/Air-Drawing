# 🎨 AI Air Drawing

## Overview
AI Air Drawing is a computer vision project that allows users to draw on a virtual canvas using hand gestures. It uses MediaPipe for hand tracking and OpenCV for real-time video processing.

## Features
- ✋ Hand gesture detection
- 🎨 Air drawing on a virtual canvas
- 🌈 Multiple color selection
- 🧽 Eraser mode
- 💾 Save drawing
- 🗑️ Clear canvas
- 🖌️ Brush tool
- 📷 Real-time webcam support

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## Installation

```bash
pip install opencv-python mediapipe numpy
```

## Run

```bash
python main.py
```
